# png cutter

### Usage 

```shell 
$ npm install png-cutter
```

```javascript
import { pngCutter } from 'png-cutter';
/**
 * @param (string) dataUrl Picture URL or Base64
 */
const result = await pngCutter(dataUrl);
```
